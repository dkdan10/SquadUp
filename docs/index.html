SquadUp

A real-time Meetup clone built with Rails, React, and Action Cable.

2019 capstone · modernized 2026 · by Daniel Keinan

Live demo → Source on GitHub

Demo login: demo@user.com / password

What it is

Users sign up against a city, create or join interest-based groups, organize events with map-pinned locations and RSVPs, and DM each other in real time. The chat layer runs over Rails Action Cable WebSockets backed by Redis.

Stack

  • Backend
    Ruby on Rails 5.2 · PostgreSQL · Redis · Action Cable
  • Frontend
    React 16 · Redux 4 · React Router 5 · Webpack 4
  • Auth
    BCrypt password digest · cookie-persisted sessions
  • Integrations
    Google Places autocomplete · Airbnb react-dates
  • Hosting
    Render (web + Postgres + Redis Blueprint)
  • CI
    GitHub Actions: Ruby 2.6 · Node 10 · Postgres 13

Features worth pointing at

  • 5-step group creation wizard. Sibling React components share state through a parent reducer; Auth Routes block edits to groups the user doesn’t own.
  • Real-time chat. Two parallel Action Cable connections per user: one streams messages for the open conversation, the other broadcasts new-channel notifications.
  • Debounced search. Client-side filter with server-side ILIKE queries on groups and events, parameterized to defeat SQL injection (covered by integration tests).
  • Maps integration. Google Places autocomplete on event creation; lat/lng persisted alongside addresses.

What I’d do differently in 2026

Reading my own 2019 code with senior eyes was the most useful part of this relaunch. Things I shipped in this Fix First branch:

  • Closed an XSS vector in the layout where html_safe bootstrapped a user-controlled JSON payload directly into a <script> tag.
  • Fixed an Action Cable mount-path bug that silently broke production messaging (mounted under /api/cable while the client connected to /cable).
  • Unbroke bundle install by replacing the yanked mimemagic 0.3.10 via marcel ~> 1.0, and bumped to Rails 5.2.8.1 for backported security patches.
  • Added GitHub Actions CI and a Render Blueprint so the deploy story is reproducible.
  • Wrote regression tests for the XSS fix and the cable mount path.

The longer modernization track — Ruby 3.3, Rails 7, React 18 + hooks, replacing jQuery $.ajax with fetch, swapping Webpack for Vite — is the next branch.
