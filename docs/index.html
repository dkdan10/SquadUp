SquadUp

A real-time Meetup clone built with Rails, React, and Action Cable.

2019 capstone · relaunched 2026 on Rails 6.1 + Ruby 3.1 · by Daniel Keinan

Live demo → Source on GitHub

Demo login: demo@user.com / password

What it is

Users sign up against a city, create or join interest-based groups, organize events with map-pinned locations and RSVPs, and DM each other in real time. The chat layer runs over Rails Action Cable WebSockets backed by Redis.

Stack

  • Backend
    Ruby 3.1 · Rails 6.1 · PostgreSQL · Redis · Action Cable
  • Frontend
    React 16 · Redux 4 · React Router 5 · Webpack 4
  • Auth
    BCrypt password digest · cookie-persisted sessions
  • Integrations
    Google Places autocomplete · Airbnb react-dates
  • Hosting
    Render Blueprint (web + Postgres + Redis)
  • CI
    GitHub Actions: Ruby 3.1 · Node 10 · Postgres 13

Features worth pointing at

  • 5-step group creation wizard. Sibling React components share state through a parent reducer; Auth Routes block edits to groups the user doesn’t own.
  • Real-time chat. Two parallel Action Cable connections per user: one streams messages for the open conversation, the other broadcasts new-channel notifications.
  • Debounced search. Client-side filter with server-side ILIKE queries on groups and events, parameterized to defeat SQL injection (covered by integration tests).
  • Maps integration. Google Places autocomplete on event creation; lat/lng persisted alongside addresses.
